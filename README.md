# Definition
The gamma function, Γ(α), for α > 0, is defined as
Γ(α) = Z 

The gamma function satisfies the recursive property
Γ(α) = (α − 1)Γ(α − 1)

When α = n and n is a positive integer, then the gamma function is related to the factorial function:
Γ(n) = (n − 1)!
